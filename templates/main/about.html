This website was developed as part of a laboratory assignment for the CI/CD course.

The goal of the project was to explore continuous integration and continuous deployment principles by building and deploying a modern web application. Django was used as the main web framework due to its simplicity and robustness. Throughout the process, we incorporated version control, automated testing, and deployment pipelines to simulate real-world software development workflows. This project serves as both a technical exercise and a demonstration of the power and efficiency that CI/CD practices bring to modern web development.

Technology used

Django is a high-level Python web framework that enables rapid development of secure and maintainable websites. It follows the "batteries-included" philosophy, meaning it comes with many built-in features like authentication, admin interface, and ORM. Django encourages clean, pragmatic design and follows the MVC (Model-View-Controller) architectural pattern, though it refers to it as MTV (Model-Template-View). It is widely used for building everything from simple websites to large-scale web applications.
